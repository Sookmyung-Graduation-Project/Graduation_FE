## 🧩 기능 요약

### ✅ 핵심 기능

1. **회원가입 및 로그인 (Firebase Auth)**
2. **프로젝트 생성 및 팀원 초대**
3. **할 일(Task) 등록, 수정, 삭제**
4. **캘린더 기반 일정 관리**
5. **산출물 파일 업로드 및 관리**
6. **팀원 피드백 작성 및 조회**
